Laravel-2FA
=================================

This is a project that implement Two-Factor Authentication(2FA) with laravel 8.

I use `Google Authenticator App` for 2FA auth.

### Requirements
- PHP >= 7.2
    - imigack
- MYSQL >= 5.7.24
- Apache >= 2.4.35

### Installation

```shell script
$ npm install --production
$ composer install
```
